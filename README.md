# Oncorhynchus-masou-formosanus-detection
偵測櫻花鉤吻鮭

利用自動化辨識及追蹤去計算櫻花鉤吻鮭的平均個數以及移動趨勢

dataset: 6000張（1000手動label 5000利用資料增強翻轉）

## Environment

tracking_result:
Python version==3.7  


run tracking_result.py:

```
python tracking_result.py
```
